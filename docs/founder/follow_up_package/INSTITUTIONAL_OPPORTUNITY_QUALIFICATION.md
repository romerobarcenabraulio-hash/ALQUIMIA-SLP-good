# Institutional Opportunity Qualification

**Uso:** matriz para decidir si una oportunidad avanza, avanza con restricciones, requiere estudio, requiere revision, decision founder o no avanza.

## Matriz de calificacion

| Criterio | Bajo riesgo | Riesgo medio | Riesgo alto |
|---|---|---|---|
| Disponibilidad de datos | Fuentes con fecha, unidad y responsable. | Datos parciales. | Sin fuentes o datos contradictorios. |
| Estudio local | Existe y esta validado. | Existe pendiente de revision. | No existe y el claim depende de el. |
| Apertura a diagnostico | Aceptan advertencias. | Quieren resultados rapidos. | Exigen oficialidad inmediata. |
| Riesgo politico | Sponsor claro y tiempos razonables. | Cabildo dividido o presion publica. | Quieren usar ALQUIMIA para legitimar decision cerrada. |
| Riesgo juridico | Juridico participa. | Juridico pendiente. | Rechazan revision juridica. |
| Concesionario | Coopera con datos. | Datos incompletos. | Bloquea informacion critica. |
| Validacion humana | Responsable identificado. | Responsable por definir. | Nadie valida. |
| Urgencia operativa | Compatible con diagnostico/piloto. | Plazo apretado. | Quieren saltar evidencia. |
| Fit Plataforma 0 | Tenant/gates/capabilities claros. | Requiere configuracion. | No aceptan gates humanos. |
| Fit vertical municipal | RSU municipal claro. | Mixto o regional. | Mezcla municipio/ZM sin control. |
| Fit empresa/institucion | Rol y datos claros. | Rol en definicion. | Busca claims reputacionales sin evidencia. |
| Field study | Se acepta como condicion. | Se discute costo/tiempo. | Lo rechazan pero quieren claim local. |
| Claim no sustentado | No hay presion. | Algunas frases riesgosas. | Exigen promesa no defendible. |

## Clasificacion

| Resultado | Condicion |
|---|---|
| Avanzar | Datos minimos, responsable humano y limites aceptados. |
| Avanzar con restricciones | Hay datos parciales; claims quedan preliminares. |
| Requiere estudio local | Claim critico depende de evidencia de campo. |
| Requiere revision juridica | Hay Cabildo, contrato, reforma, dictamen o documento institucional. |
| Requiere founder decision | Riesgo comercial/metodologico alto pero corregible. |
| No avanzar | Exigen claims no sustentados, ocultar brechas o automatizar aprobaciones. |

## Regla de calificacion

Una oportunidad no avanza a propuesta si depende de benchmark como estudio local, inferencia como dato validado, estimacion como oficial, capa de aprendizaje supervisado/flujo de borradores asistidos como aprobador, o mezcla municipio/ZM.
