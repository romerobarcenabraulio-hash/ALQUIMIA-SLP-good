# Data Request Packets

**Uso:** paquetes de solicitud de datos por interlocutor despues de una reunion.

## Municipio

| Categoria | Detalle |
|---|---|
| Datos minimos | Municipio, area responsable, reglamento, contratos vigentes, contacto tecnico. |
| Datos deseables | Actas, reportes, mapas, antecedentes y metas. |
| Formato | PDF/CSV/XLSX con fuente y fecha. |
| Periodicidad | Inicial y actualizacion mensual si hay piloto. |
| Fuente responsable | Secretaria/servicios publicos/area designada. |
| Validacion humana | Responsable municipal y founder. |
| Se puede calcular | Diagnostico preliminar y brechas. |
| No se puede afirmar sin estudio local | Generacion/composicion real, impacto o captura oficial. |
