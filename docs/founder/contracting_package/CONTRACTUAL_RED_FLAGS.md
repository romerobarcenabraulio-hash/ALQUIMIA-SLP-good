# Contractual Red Flags

**Uso:** banderas rojas antes de propuesta o contrato.

| Red flag | Riesgo | Accion recomendada | Condicion de bloqueo | Escalamiento |
|---|---|---|---|---|
| Cliente exige garantia de ahorro. | Promesa no sustentada. | Reformular como escenario con supuestos. | Si exige garantia contractual. | Founder/juridico. |
| Cliente quiere presentar benchmark como dato local. | Falsa evidencia municipal. | Requerir estudio local. | Si lo exige para Cabildo/contrato. | Founder/revision metodologica. |
| Cliente pide ocultar brecha critica. | Riesgo reputacional y tecnico. | Mantener advertencia. | Si condiciona firma a ocultarla. | Founder. |
| Cliente quiere usar estimacion como dato oficial. | Oficialidad falsa. | Degradar a estimacion. | Si se niega a advertencia. | Juridico/founder. |
| Cliente no acepta validacion humana. | Gobernanza invalida. | Explicar gates humanos. | Si exige aprobacion automatica. | Founder. |
| Cliente quiere que capa de aprendizaje supervisado apruebe. | Autoridad falsa. | Aclarar que capa de aprendizaje supervisado observa/sugiere. | Si insiste. | revision metodologica/founder. |
| Cliente no puede proveer datos minimos. | Diagnostico debil. | Diagnostico preliminar con brechas. | Si exige conclusion oficial. | Founder. |
| Concesionario bloquea informacion. | Datos operativos incompletos. | Mesa tecnica/solicitud formal. | Si el claim depende de esos datos. | Municipio/founder. |
| Municipio/ZM se mezclan. | Conclusion territorial invalida. | Separar fuentes y claims. | Si no se puede separar alcance. | validacion de arquitectura/founder. |
| No hay responsable humano. | Nadie valida. | Designar responsable. | Si no existe decisor. | Founder/cliente. |
| No hay opt-in para aggregate. | Privacidad. | Mantener datos fuera de analytics agregada. | Si piden benchmark identificable. | Founder/revision metodologica. |

## Regla final

Una red flag no siempre mata la oportunidad. Pero si el cliente exige conservar una promesa no sustentada, la propuesta debe pausarse o rechazarse.
