## Capítulos del simulador (4 + rubros)

Los cuatro capítulos pedagógicos y sus rubros están definidos en la Constitución del frontend. Este directorio aloja **salidas documentales**, no la configuración UI.

| Capítulo | Pregunta guía | Rubros |
|----------|---------------|--------|
| 1 · Diagnóstico | ¿Cuál es la situación? | Ambiental · Social · Gobernanza · Normativo · Financiero · Teoría de cambio |
| 2 · Planificación | ¿Qué haremos? | Estratégico · Operativo · Económico |
| 3 · Modelo | ¿Cómo se sostiene? | Institucional · Financiero · Gobernanza |
| 4 · Control | ¿Está funcionando? | Cumplimiento · Monitoreo · Reporteo |
